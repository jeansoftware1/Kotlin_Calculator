**Projeto inicial para a Calculadora Kotlin**

Versão inicial do aplicativo Android  Calculadora Kotlin.
Principais recursos e arquivos incluídos:
-Lógica principal do aplicativo: O arquivo MainActivity.kt contém a lógica fundamental para uma calculadora básica, incluindo o tratamento de entradas de dígitos, operadores (+, -, *, /), limpeza e igualdade. A interface do usuário utiliza ViewBinding.

- Interface do usuário: Um layout em activity_main.xml é definido usando um LinearLayout e um TableLayout para estruturar a exibição e os botões da calculadora. Uma imagem de fundo e estilos personalizados para botões transparentes estão incluídos.
